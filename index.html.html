2. Machine-Speed Containment Automation Script

The code block below outlines the core, non-destructive logic executed by the XDR agent during an incident. This automation stops lateral propagation across isolated networks:

containment_policy.py // XDR Integration Endpoint Python 3.10 
import requests
import sys

# API credentials automatically resolved from runtime environment
XDR_API_ENDPOINT = "https://api.xdr-shield.local/v2/isolate"
CONTAINMENT_TOKEN = "ENC_SEC_K8s_VAL_TOKEN"

def trigger_isolate(host_ip, threat_id):
    headers = {
        "Authorization": f"Bearer {CONTAINMENT_TOKEN}",
        "Content-Type": "application/json"
    }
    payload = {
        "target_ip": host_ip,
        "incident_id": threat_id,
        "isolation_level": "Strict",
        "terminate_active_sessions": True
    }
    
    try:
        response = requests.post(XDR_API_ENDPOINT, json=payload, headers=headers, timeout=10)
        if response.status_code == 200:
            print(f"[SUCCESS] Host {host_ip} isolated successfully.")
            return True
        else:
            print(f"[ERROR] Containment API error: {response.status_code}")
            return False
    except Exception as e:
        print(f"[CRITICAL] Network Exception: {str(e)}")
        return False
